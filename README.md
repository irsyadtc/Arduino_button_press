############################################################
#	Arduino button press example
############################################################

## Description

This sketch demonstrates an example of detecting different mode of button press such as single, double, and hold.

## Setup
 
 Microcontrollers:
 - Mega 2560

 Components:
 - button
 - 74HC14
 - 10kOhm x 2 resistor
 - 0.1uF capacitor


 Mega 2560
 
 Pin connections
 - 29 74HC14 1Y (Button reset)
 - Vin 5V supply
 - GND GND
